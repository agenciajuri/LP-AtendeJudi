# AtendeJuri Landing Page

Landing page independente em Next.js para a marca **AtendeJuri**.

## Como rodar

```bash
npm install
npm run dev
```

## Placeholders para preencher depois

No arquivo `app/page.tsx`, ajuste:

- `#WHATSAPP_LINK`
- `#DEMO_LINK`
- `#FORM_ACTION`

## Direção visual

A página segue o sistema visual de `../prompt-ui-agencia-juri.md`: visual jurídico editorial, cores marinho/dourado/marfim, tipografia Fraunces + Inter, linguagem simples e sem estética futurista.
